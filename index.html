QuotelyMint API

Simple API with creative quotes

⚖️ MIT Licensed

Docs :
https://quotelymint.ml/quote - Random Quote
https://quotelymint.ml/author - Get all authors
https://quotelymint.ml/author?authorname - Get quotes from a specific author ( IN DEV )
https://quotelymint.ml/categories - Get all categories
https://quotelymint.ml/nameofcategory - Get a random quote from the category
